# ต่อไปผมจะทำเป็น แอปนึงหรือ ฟังก์ชันนึงดีครับ เพราะว่ามันเป็นส่วนนึงของการนับสินค้าด้วยอะครับ

แต่อันนี้เป็นการทำงานประมาณว่า

1. เมื่อมันจะมีโปรแกรมนึงของเขาแล้วเขาจะต้อง ตรวจสอบสถานะการส่งของว่าของถึงไหนแล้วพนะงานส่งรูปเข้าไป ในไลน์หรือยัง เหมือนเหมือนคล้ายโปรแกรมของที่ผมกำลังจะไปเชื่อมต่อของเขา ไม่รู้ว่ามันจะยุ่งยากไหมครับหมายความว่าผมอาจจะเอาบางสวิ่งบางอย่างที่ผมอาจจะไปเชื่อม API แต่ผมก็อาจจะทำเองก็ได้แต่อันนี้มันจะไปเชื่อมกับระบบคลังสินค้าเขาน่ะครับ

2. เอาเป้นวาตอนนี้ผมอยากได้ประมาณว่า เมื่อเขา tracking สินค้า สถานะ ว่าอยู่ไหน แล้วในแอปที่เราจะทำก็แค่ แต่ก่อนที่จะบอก พนักงานถ่ายรูปในไลน์ครับ ซึ่งผมมองว่า กว่าเขาระรวบรวมรูปภาพแลเวเขาก็ไปอัปเดทในระบบอีกทีผมมองว่ามันซ้ำซ้อน และอาจจะทำให้พนักงานเหนื่อย ผมก็เลยอยากให้เขาแนบรูปภาพบอกว่า สาขานี้มีสินค้ามาส่งนะ ของพนักงานคนนี้นะที่มาส่ง แล้วก็ให้ถ่ายยืนยันว่าสินค้าส่งมาแล้ว พร้อมกับแนบว่า ได้รับสินค้าเข้าสาขาแล้วอะไรประมาณนี้อ่ะครับ เราจะทำเว็ปเป็นตัวอย่างให้เขาดูก่อนก็ได้ครับ

3. เช็คชื่อพนักงาน ทุกๆวันพนักงานจะต้องเข้าทำงาน แล้วต้องถ่ายบอกว่า ตอนนี้เข้างานแล้ว เขาจะต้องจ่ายรูปที่เขายืนกับบูท ที่เขาขายอาหารเสิริมหรือผลิตพันของเขาน่ะครับ
