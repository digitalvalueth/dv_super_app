# คู่มือผู้ใช้งาน FITT BSA — สำหรับทีม Phithan

อัปเดต: 27 เมษายน 2025

---

## 1. ภาพรวมระบบ

FITT BSA แบ่งผู้ใช้งานออกเป็นหลายบทบาท แต่ละบทบาทเห็นเมนู / ทำงานได้แตกต่างกัน

| บทบาท           | ชื่อในระบบ    | หน้าที่หลัก                                |
| --------------- | ------------- | ------------------------------------------ |
| พนักงานขาย (BA) | `employee`    | นับสต็อก, เช็คชื่อเข้า-ออกงาน, รับสินค้า   |
| หัวหน้างาน      | `supervisor`  | ดูแล BA ของตน, อนุมัติการนับ, ดูรายงานสาขา |
| ผู้จัดการ       | `manager`     | ดูแลหลายสาขา, อนุมัติการนับ, จัดการพนักงาน |
| ผู้ดูแลบริษัท   | `admin`       | จัดการบริษัท, สาขา, ผู้ใช้, รายงานทั้งหมด  |
| ผู้ดูแลระบบ     | `super_admin` | ข้ามบริษัท, ตั้งค่าระดับระบบ               |

---

## 2. การเข้าใช้งานครั้งแรก

1. **รับอีเมลคำเชิญ** จากผู้ดูแล (Subject: "คุณได้รับเชิญเข้าใช้งาน FITT BSA")
2. **คลิกลิงก์** ในอีเมล → จะเปิดเว็บ/แอป
3. **เข้าสู่ระบบด้วย Google / Apple** ใช้อีเมลเดียวกับที่ได้รับเชิญ
4. ระบบจะกำหนดบทบาท, สาขา, รหัส BA โดยอัตโนมัติ

---

## 3. เมนูสำหรับ BA (พนักงานขาย)

### บนแอปมือถือ

- **หน้าแรก** — แดชบอร์ดส่วนตัว
- **เช็คชื่อเข้า/ออก** — ถ่ายรูปพร้อม watermark + GPS
- **นับสต็อก** — ถ่ายรูปสินค้า, AI นับให้, แก้ไขจำนวนได้
- **รับสินค้า** — บันทึกการรับสินค้าใหม่
- **การตั้งค่า** — โปรไฟล์, ประวัติเข้าระบบ, ความเป็นส่วนตัว

> หมายเหตุ: BA จะเห็นเฉพาะข้อมูลของสาขาตนเท่านั้น

---

## 4. เมนูสำหรับ Supervisor / Manager

### บนเว็บ Admin (`/stock-counter/dashboard`)

- **แดชบอร์ด** — ภาพรวมตัวเลข
- **Supervisor Dashboard** — ดู BA ในสายงาน
- **สาขา** — ดูรายการสาขาที่ดูแล
- **ผู้ใช้งาน** _(manager เท่านั้น)_
- **เชิญผู้ใช้** _(admin เท่านั้น)_
- **นำเข้าผู้ใช้ (Excel)** _(admin เท่านั้น)_
- **นำเข้าสาขา (Excel)** _(admin เท่านั้น)_
- **สินค้า / รอบการนับ / ข้อมูลการนับ / สรุปการนับ**
- **เช็คชื่อพนักงาน / ตั้งค่าเวลาทำงาน**
- **แจ้งเตือนขาดงาน**
- **ค่าคอมมิชชั่น** _(manager+)_
- **โลจิสติกส์ / รับสินค้า**
- **รายงาน**

## 6. ขั้นตอนทั่วไปประจำวัน (BA)

### 6.1 เช็คชื่อเข้างาน

1. เปิดแอป → เลือก **เช็คชื่อเข้า/ออก** → **เช็คชื่อเข้า**
2. เลือกกะ (ถ้ามี) → เลือกสาขา (ถ้ามีหลายสาขา)
3. ถ่ายรูปตนเอง — ระบบใส่ watermark (ชื่อ, เวลา, พิกัด)
4. กด **ยืนยัน** → เสร็จ
5. หากอยู่ห่างจากสาขาเกินรัศมีที่ตั้งไว้ ระบบจะเตือน (แต่ไม่บล็อก)

### 6.2 นับสต็อก

1. **นับสต็อก** → เลือกสินค้าที่ได้รับมอบหมาย
2. กรอกจำนวนก่อนนับ → ถ่ายรูปสต็อก
3. รอ AI นับ → ตรวจสอบจำนวน
4. หากตรง → กดยืนยัน, หากต่าง → ระบุสาเหตุ
5. หัวหน้าจะอนุมัติในระบบเว็บ

### 6.3 เช็คชื่อออก

ทำเหมือนข้อ 6.1 แต่เลือก **เช็คชื่อออก**

## 7. การตั้งค่าและการลบบัญชี

- **การตั้งค่า → ความเป็นส่วนตัวและข้อมูล** — ดูประวัติเข้าระบบ, ขอลบบัญชี
- การลบบัญชีต้อง:
  1. ติ๊กยอมรับว่าไม่สามารถกู้คืนได้
  2. พิมพ์คำว่า `DELETE`
  3. กดปุ่มลบ → ยืนยันอีกครั้งใน popup
- หลังลบ ข้อมูลทั้งหมดจะถูกลบออกจากระบบทันที
