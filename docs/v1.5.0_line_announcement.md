📢 **อัปเดตระบบ Stock Counter & Admin Dashboard เวอร์ชันใหม่ (v1.5.0)** 🎉
อัปเดตฟีเจอร์ใหม่เพื่อเพิ่มประสิทธิภาพในการจัดการทีมงาน และเพิ่มความแม่นยำในการตรวจนับสต็อกผ่านมือถือครับ!

🏢 **1. ยกระดับการจัดการสาขาและทีมงาน (ระบบ Web Admin)**
* **เพิ่มข้อมูลหมวดหมู่ & หัวหน้างาน:** ตอนนี้แอดมินสามารถระบุ `Seller Category` (เช่น Lotus, BigC, Watson) และ `Supervisor` (หัวหน้าพนักงาน) ให้กับแต่ละสาขาและพนักงานได้แล้ว ช่วยให้ค้นหาและจัดกลุ่มได้ง่ายขึ้น
* **ดูข้อมูลสาขาง่ายขึ้น:** เปลี่ยนรูปแบบหน้าจัดการสาขาเป็น "ตาราง" (Table) ทำให้มองเห็นข้อมูลสำคัญทั้งหมด (ชื่อ, รหัส, Geofence, จำนวนพนักงาน) ได้ครบจบในหน้าเดียว
* **การจัดการสิทธิ์พนักงาน:** ในหน้า "เชิญผู้ใช้" (Invitations) แอดมินสามารถกด **เปิด/ปิด การใช้งานบัญชี (ระงับสิทธิ์)** ของพนักงานได้ทันที พร้อมแสดงชื่อ-สกุล และรหัสพนักงาน (BA Code) ให้ตรวจสอบได้ง่ายขึ้น

📱 **2. เพิ่มความรัดกุมและความแม่นยำ (ระบบ Mobile App)**
* **บังคับถ่ายรูปจริงหน้างานเท่านั้น:** ในฟีเจอร์ "รับสินค้า" นำปุ่มเลือกรูปจากคลังออก เพื่อให้พนักงานต้องถ่ายรูปสถานที่/สินค้าจริง ณ เวลานั้นเท่านั้น (ป้องกันการใช้รูปเก่า)
* **จำกัดระยะการเช็คอิน (Geofence):** เพื่อความแม่นยำในการลงเวลาทำงาน แอดมินสามารถตั้งรัศมีพิกัดสาขาได้สูงสุดไม่เกิน 500 เมตร (แนะนำที่ 200 เมตร) เพื่อป้องกันการเช็คอินหรือนับสต็อกนอกพื้นที่สาขา
* **แก้บั๊กรูปภาพ:** แก้ไขปัญหารูปภาพสินค้านับสต็อกไปโผล่ข้ามสาขา (หากเป็นสินค้า SKU เดียวกัน) ตอนนี้รูปจะแยกตามสาขาอย่างถูกต้อง 100%
