## หมายเหตุพิเศษสำหรับ Phithan

- **BA Code** — รหัสประจำตัวพนักงาน Phithan ใช้ในการ map กับระบบ HR
- **Seller** — ระบุชื่อบริษัท/แบรนด์ที่พนักงานสังกัด (เช่น "Phithan", "PT Group")
- **Supervisor mapping** — Manager เป็นผู้ทำการ map BA → Supervisor ผ่านหน้าเชิญผู้ใช้
- **Geofence** — Admin ตั้งพิกัดสาขา + รัศมี (ค่า default 200 เมตร) ระบบเตือนเมื่อ BA อยู่นอกพื้นที่ขณะถ่ายรูป
