# แนวทางการถ่ายรูปสินค้าให้ AI วิเคราะห์ได้แม่นยำ

อัปเดต: 27 เมษายน 2025

ระบบใช้ Google Gemini AI ในการนับจำนวนสินค้าจากรูปถ่าย หากถ่ายรูปดี ความแม่นยำสามารถทะลุ 95%+ ได้

---

## ✅ สิ่งที่ควรทำ

### 1. แสงสว่างเพียงพอ

- ถ่ายในที่ที่มีแสงสว่างพอ (เปิดไฟร้านทุกดวง / ใช้ไฟสปอตไลต์)
- หลีกเลี่ยงเงาทับสินค้า
- หากแสงไม่พอ ใช้ Flash จากแอปได้

### 2. มุมถ่าย

- ถ่ายตรง ๆ จากด้านหน้า (90°) หรือมุมเฉียงเล็กน้อย (~15°)
- ให้สินค้าทุกชิ้นอยู่ในเฟรม
- ระยะห่างพอเหมาะ — ไม่ใกล้จนเห็นรายละเอียด blur และไม่ไกลจนนับไม่ออก

### 3. การจัดวางสินค้า

- จัดเรียงเป็นแถว / กอง ให้เห็นชัดเจน
- หากซ้อนหลายชั้น ถ่ายแต่ละชั้นแยกกัน
- ไม่ให้สิ่งของอื่นบัง (มือ, ป้ายราคา, เศษกระดาษ)

### 4. โฟกัส

- รอให้กล้องโฟกัสชัดก่อนกดถ่าย
- หากใช้แอปแล้วเบลอ ให้กดถ่ายใหม่
- ถือมือนิ่ง — กลั้นหายใจสักวินาที

### 5. รวมสินค้าทั้งหมดในเฟรมเดียว

- ถ่ายให้ครบทุกชิ้น
- หากชั้นวางยาว แบ่งถ่ายเป็น 2 รูปได้ (แต่จะต้องเลือกรูปที่ดีที่สุดส่งเข้าระบบ)

---

## ❌ สิ่งที่ไม่ควรทำ

| ปัญหา                       | ผลกระทบ                       |
| --------------------------- | ----------------------------- |
| ถ่ายในที่มืด / ย้อนแสง      | AI นับผิด, ความเชื่อมั่นต่ำ   |
| ภาพเบลอ / สั่น              | นับขาด-เกิน                   |
| สินค้าซ้อนทับเยอะ           | นับเฉพาะชั้นบน                |
| มีคน/มือ/สิ่งของบัง         | นับขาด                        |
| ถ่ายในมุมหักเฉียงมาก (>45°) | AI สับสน                      |
| ใช้รูปจากแกลอรี่ที่ตัดต่อ   | ระบบจะปฏิเสธ (ตรวจ watermark) |

---

## 📷 ตัวอย่างกรณีพิเศษ

### สินค้าใส (ขวดแก้ว, แชมพู)

- ใช้พื้นหลังทึบสีตัด (ผนังขาว / กระดาษดำ)
- หลีกเลี่ยงสะท้อนแสงแบบจ้า

### สินค้าเล็ก (ลิปสติก, ตัวอย่าง)

- ถ่ายใกล้ขึ้น
- ใช้ฉากหลังเรียบ
- จัดเรียงเป็นกริด

### สินค้าซ้อนกอง (เช่นกล่องเครื่องดื่ม)

- ระบุจำนวนชั้นในช่องหมายเหตุด้วย
- หรือถ่ายจากด้านข้างให้เห็นความสูง

---

## 🔍 หลังถ่าย

1. ดูตัวอย่างก่อนกดส่ง
2. ตรวจดูว่า watermark (ชื่อ, เวลา, พิกัด) ปรากฏชัด
3. หากภาพไม่ดี กด **ถ่ายใหม่**
4. ส่งไป AI วิเคราะห์
5. ตรวจสอบจำนวนที่ AI นับ — หากต่างจริง ให้แก้ไขจำนวนพร้อมระบุสาเหตุ

---

## 📊 ค่าความเชื่อมั่น (AI Confidence)

| ค่า     | ความหมาย       | ควรทำอย่างไร           |
| ------- | -------------- | ---------------------- |
| 90-100% | นับชัดมาก      | ใช้ตามผลได้เลย         |
| 70-89%  | ค่อนข้างมั่นใจ | ตรวจตา 1 รอบ           |
| 50-69%  | ไม่ค่อยมั่นใจ  | นับมือเทียบ            |
| < 50%   | นับยาก         | ถ่ายใหม่ในแสงที่ดีกว่า |

---

## 💡 เคล็ดลับเพิ่มเติม

- ถ่ายในเวลาเดียวกันของวัน (เช้า/บ่าย) เพื่อแสงคงที่
- ทำความสะอาดเลนส์มือถือก่อนถ่าย
- ตั้งค่ากล้องเป็น "HDR" หากมือถือรองรับ
- หากมีปัญหาบ่อย ติดต่อ Manager ขอเปลี่ยนตำแหน่งจัดวางสินค้า
