📢 **อัปเดตระบบ FITT BSA Platform เวอร์ชัน v1.5.1 (Web v1.5.1)** 🎉
ปรับปรุงระบบสิทธิ์การใช้งาน, นำเข้าข้อมูล Excel และปรับโครงสร้าง Role ให้เหมาะสมมากขึ้นครับ!

---

🔐 **1. ปรับปรุงระบบสิทธิ์การเข้าถึง Module (Access Control)**
* **บล็อก Employee จากเว็บ:** พนักงานระดับ Employee จะไม่สามารถเข้าใช้งานเว็บแอดมินได้ ต้องใช้งานผ่าน Mobile App เท่านั้น
* **Admin เห็นเฉพาะ Module ที่ได้รับสิทธิ์:** Admin จะเห็นเฉพาะ Module ที่ตัวเองถูกเพิ่มใน whitelist (ไม่ bypass อัตโนมัติอีกต่อไป) → เฉพาะ Super Admin เท่านั้นที่เห็นทุก Module
* **Admin จัดการสิทธิ์ให้คนอื่นได้:** Admin สามารถเปิด/ปิดสิทธิ์ Module ให้ผู้ใช้ที่ role ต่ำกว่า (Supervisor, Manager, Employee) ได้จากหน้า "จัดการสิทธิ์"
* **ป้องกันการแก้ไข Role สูงกว่า:** Admin ไม่สามารถจัดการสิทธิ์ของ Admin คนอื่นหรือ Super Admin ได้ (แสดงไอคอน 🔒)

👥 **2. ปรับโครงสร้าง Role**
* **รวม Staff กับ Employee:** เดิมมี 2 role ที่ทำงานเหมือนกัน (Staff / Employee) → ตอนนี้เหลือแค่ **Employee** อย่างเดียว
* **เปลี่ยนชื่อ Manager → Supervisor:** เมนู "จัดการ Manager" เปลี่ยนเป็น "จัดการ Supervisor" พร้อม query ที่รองรับทั้ง supervisor และ manager role
* **เพิ่มค้นหาสาขา:** Modal เลือกสาขาใน Supervisor มีช่องค้นหาได้แล้ว

📊 **3. ปรับปรุงระบบนำเข้า Excel**
* **Branches Template:** เพิ่มคอลัมน์ `seller` ในเทมเพลตนำเข้าสาขา → บันทึก `sellerCategory` ลง Firestore อัตโนมัติ
* **แก้ BA Code เลข 0 นำหน้า:** แก้ไขปัญหา BA Code เช่น `0087` ถูกตัดเหลือ `87` โดย:
  - อ่าน formatted text จาก raw cell ของ Excel
  - ตั้ง format คอลัมน์ baCode ในเทมเพลตเป็น Text (`@`)
